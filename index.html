PHP


A linguagem de programação PHP é do tipo interpretada e é, atualmente, uma das mais usadas no mundo.
O PHP é vastamente usado para criar aplicações e sistemas, do lado do servidor.
Usa-se muito o PHP para criar páginas com conteúdo de uma maneira dinâmica, além de ser possível acessar
e interagir com servidores e banco de dados.
